Features destacadas

🔥

Firebase completo

Auth (email + Google), Firestore para productos/órdenes, Storage para imágenes

🌓

Dark / Light mode

Toggle en perfil, persistido con DataStore Preferences

🌐

Bilingüe ES/EN

Localización completa, idioma persistido en DataStore

⚡

Skeleton loading

Shimmer UI mientras cargan los productos — sin pantallas vacías

🏗️

MVVM estricto

ViewModels en MainActivity, StateFlow, sin LiveData, sin estados huérfanos

🛡️

Fallback de datos

Si Firestore falla, usa SampleData local — nunca pantalla vacía

📲 Cómo instalar el APK

1
Descargá el archivo radiance-release.apk desde GitHub Releases
2
En tu Android: Ajustes → Seguridad → Instalar apps de fuentes desconocidas (o permitir desde el navegador que descargaste el APK)
3
Abrí el archivo .apk descargado y tocá Instalar
4
Requiere Android 8.0 o superior (API 26+) — compatible con la gran mayoría de dispositivos Android actuales

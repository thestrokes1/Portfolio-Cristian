✨ Radiance

App Android de indumentaria e-commerce — explorá, filtrá y comprá ropa con una experiencia de usuario moderna construida con Jetpack Compose y Material Design 3.

⬇ Descargar APK ⌨️ GitHub

Pantallas de la app

✨
Splash & Onboarding
Animación de entrada + 3 slides de presentación con swipe o skip
🔐
Login / Registro
Email + contraseña y Google Sign-In via Firebase Auth
🏠
Home
Grid de productos, filtros por categoría, sort por precio y rating, pull-to-refresh
👗
Detalle de Producto
Galería de imágenes, selector de talle/color/cantidad, agregar al carrito o wishlist
🔍
Búsqueda
Búsqueda en tiempo real por nombre, marca y categoría. Sugerencias recientes
🛒
Carrito
Gestión de ítems, ajuste de cantidad, cálculo de subtotal, descuentos y total
❤️
Wishlist
Grilla de productos guardados, agregar/quitar con un tap
📦
Checkout
3 pasos: dirección de envío → resumen → confirmación con orden guardada en Firestore
👤
Perfil
Stats de cuenta, dark mode toggle, idioma ES/EN, configuración y logout

Features destacadas

🔥

Firebase completo

Auth (email + Google), Firestore para productos/órdenes, Storage para imágenes

🌓

Dark / Light mode

Toggle en perfil, persistido con DataStore Preferences

🌐

Bilingüe ES/EN

Localización completa, idioma persistido en DataStore

⚡

Skeleton loading

Shimmer UI mientras cargan los productos — sin pantallas vacías

🏗️

MVVM estricto

ViewModels en MainActivity, StateFlow, sin LiveData, sin estados huérfanos

🛡️

Fallback de datos

Si Firestore falla, usa SampleData local — nunca pantalla vacía

📲 Cómo instalar el APK

1
Descargá el archivo radiance-release.apk desde GitHub Releases
2
En tu Android: Ajustes → Seguridad → Instalar apps de fuentes desconocidas (o permitir desde el navegador que descargaste el APK)
3
Abrí el archivo .apk descargado y tocá Instalar
4
Requiere Android 8.0 o superior (API 26+) — compatible con la gran mayoría de dispositivos Android actuales
